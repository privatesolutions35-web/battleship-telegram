# Морской Бой - Telegram Web App

## 1. Project Overview

- **Название**: Морской Бой (Battleship)
- **Тип**: Telegram Web App Game
- **Суть**: Классическая игра "Морской бой" против AI-бота
- **Целевая аудитория**: Пользователи Telegram

## 2. UI/UX Specification

### Layout Structure

- **Header**: Заголовок игры, счет
- **Main Area**: Два игровых поля 10x10 (свое и противника)
- **Footer**: Кнопки управления, статус игры

### Responsive Design

- Mobile-first (Telegram Web App)
- Min-width: 320px
- Max-width: 600px (оптимально для мобильных)

### Visual Design

**Color Palette**:
- Background: `#0a1628` (темно-синий океан)
- Primary: `#1e3a5f` (морской синий)
- Accent: `#00d4ff` (бирюзовый неон)
- Hit: `#ff3b3b` (красный)
- Miss: `#4a5568` (серый)
- Ship: `#e8e8e8` (белый)
- Grid: `#2d4a6f` (сетка)
- Text: `#ffffff`

**Typography**:
- Font: `'Orbitron', sans-serif` (футуристичный)
- Title: 24px bold
- Grid cells: 10px
- Status: 16px

**Visual Effects**:
- Neon glow на акцентах
- Волны на фоне (CSS анимация)
- Анимации попаданий и промахов
- Подсветка клеток при наведении

### Graphics (открытые источники)

- Корабли: SVG иконки или CSS-формы
- Фон: CSS gradient + wave анимация
- Взрывы: CSS анимации
- Ядра: CSS круги

### Sounds (бесплатные источники)

- Выстрел: короткий звук выстрела
- Попадание: взрыв
- Промах: плеск воды
- Победа/поражение: сигнал

## 3. Functionality Specification

### Core Features

1. **Размещение кораблей**
   - 1 корабль (4 клетки)
   - 2 корабля (3 клетки)
   - 3 корабля (2 клетки)
   - 4 корабля (1 клетка)
   - Drag & drop или клик для размещения
   - Поворот корабля кнопкой

2. **Игровой процесс**
   - Ход игрока: клик по клетке противника
   - Ход бота: AI атакует случайную/умную клетку
   - Отображение результатов (попадание/промах)

3. **AI Бот**
   - Простой уровень: случайные ходы
   - Умный режим: ищет рядом при попадании

4. **Правила**
   - Корабли не соприкасаются
   - Уничтожение корабля - уведомление
   - Победа при уничтожении всех кораблей

### User Flow

1. Начальный экран → "Играть"
2. Размещение своих кораблей
3. Бой (ход игрока → ход бота)
4. Конец игры → результат

## 4. Technical Specification

- Telegram Web App API (Telegram.WebApp)
- Single HTML файл
- Vanilla JS (без фреймворков)
- LocalStorage для сохранения

## 5. Acceptance Criteria

- [ ] Игра запускается в Telegram
- [ ] Корабли размещаются корректно
- [ ] Игра против бота работает
- [ ] AI делает ходы
- [ ] Правила соблюдаются
- [ ] Есть звуки и визуальные эффекты
- [ ] Победа/поражение определяется
