# 🚢 Морской Бой - Battleship Telegram Game

Клиент-серверная игра для Telegram.

## Файлы

```
battleship/
├── index.html      # Одиночная игра (vs бот)
├── server/
│   ├── package.json
│   ├── server.js  # API сервер (Node.js)
│   └── bot.py    # Telegram бот (Python)
├── MULTIPLAYER.md # Описание мультиплеера
└── DEPLOY.md    # Инструкция запуска
```

## Быстрый старт (одиночная игра)

1. Открой `index.html` в браузере
2. Играй против AI-бота

## Мультиплеер

Для игры с другими игроками нужен сервер + Telegram бот.

См. `DEPLOY.md` для инструкции запуска.

## Возможности

- Одиночная игра vs AI
- Мультиплеер (2 игрока)
- Звуковые эффекты
- Адаптивный дизайн (мобильные)
- Telegram Web App

## Язык

- Интерфейс: English
- Бот (мультиплеер): Русский
